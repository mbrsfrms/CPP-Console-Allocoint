# CPP-Console-Allocoint

Allocoint is a C++ console-based budget allocation system that allows users to set a total monthly budget, create custom spending categories, and distribute funds while ensuring proper input validation and preventing overspending. It also provides a summary of allocations, remaining balance, and budget status.

This project is an application requirement for our Data Structures and Algorithms (DSA) course.

Features
- Set total monthly budget
- Create custom categories
- Allocate budget per category
- Input validation for safe and correct entries
- Budget summary with status check
  
Data Structures Used
- struct
- vector
  
Language Used
- C++

Purpose: This project demonstrates the application of basic data structures and control flow in solving real-life budgeting problems.
